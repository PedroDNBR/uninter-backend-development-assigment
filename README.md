### Setup:

1. Troque as credencias do banco de dados MySQL do arquivo application.properties para as do seu MySQL, não é necessário criar o banco já que a aplicação faz automaticamente
2. Importe o arquivo UNINTER - TRABALHO DE DESENVOLVIMENTO WEB BACK-END.postman_collection.json no seu postman
3. Rode a aplicação utilizando o eclipse
4. Use o postman para testar os diretórios
